How to use
Bulk Task Creator

This walkthrough covers everything from getting your API token to running your first batch of tasks. It takes about five minutes the first time — less once you know the steps.

1
Get your ClickUp API Token

Your API token is how Bulk Task Creator connects to your ClickUp account. Think of it as a key that lets the tool talk to your workspace on your behalf.

Here's where to find it:

  • Open ClickUp and click your avatar or initials in the top left corner
  • Click Settings
  • In the left sidebar, click Apps
  • Under API Token, click Copy

Your token starts with pk_ and is a long string of letters and numbers. Copy the whole thing.

Your token is private. Don't share it with anyone. Bulk Task Creator sends it directly to ClickUp from your browser — it never touches our servers.
2
Create a posting schedule

Before you create tasks, you save your repeat pattern as a "posting schedule." This is the part you set once and reuse every time — so next month you just load it and go.

On the home screen, click Create a posting schedule and fill in three things:

  • Name it — something descriptive like "Client IG Posts" or "Monday Emails." You'll see this name every time you come back.
  • Set the pattern — choose how often tasks repeat. Every day, every few days, every few weeks, or specific days of the week (like every Monday and Wednesday).
  • Name your tasks — choose how each task will be titled. You can add your own text before the date ("IG Post: March 3"), use the same title every time ("Post Topic TBD"), or just use the date on its own.
Not sure what to put for task names? Think about what you'd want to see in your ClickUp list. "Monday Email: Topic TBD" is a popular one — it tells you what type of task it is, and "Topic TBD" is a reminder to fill in the content later.

Click Save schedule when you're done. It'll show up as a card on your home screen.

3
Select your schedule and open the app

On the home screen you'll see your saved schedules as cards. Click the one you want to use — it'll highlight to show it's selected.

Then click Create tasks with this schedule to move into the run flow.

Want to skip the schedule step? You can also click "Skip — set everything manually" to configure everything from scratch each time. This is useful if you have a one-off batch that doesn't fit your usual patterns.
4
Connect to ClickUp and pick your list

This is where you paste in your API token and choose which ClickUp list you want the tasks created in.

  • Paste your API token in the first field. It saves automatically — you won't need to paste it again next time.
  • Click Browse my lists to load your ClickUp workspace.
  • Click through your workspace, space, and folder to find the right list — or use the search box to find it by name.
  • Click the list to select it. You'll see the name appear as a confirmation.

Once you select a list, the app automatically pulls in your real statuses, team members, and custom fields — so you won't need to look anything up manually.

First time loading your workspace? It might take 10-15 seconds to pull everything in, especially if you have a large ClickUp workspace. This is normal.
5
Set your dates

Pick your first task date and last task date. The app will create one task for every date that matches your schedule between those two dates.

As you fill this in, you'll see a preview showing how many tasks will be created and the date range they cover. This updates automatically so you can adjust until it looks right.

Example: If your schedule is Monday, Wednesday, Friday and you set March 1 to March 31 as your range, you'll get 13 tasks — one for every M/W/F in March.

Your schedule settings (pattern, naming) are already loaded from the schedule you saved. You can adjust anything here if you need to tweak it for this particular run.

6
Configure task settings

This step is optional — the defaults work fine for most people. But here's what you can set:

  • Due date — set the due date to match the task date, a few days before it, or leave it blank.
  • Post date custom field — if your ClickUp list has a custom "Post Date" or "Publish Date" field, turn this on and select it. The app will fill it in automatically for every task.
  • Custom fields — any other custom fields on your list appear here. Set a default value and it gets applied to every task. Great for things like "Post Type" (Social Post, Email, Blog) or "Client Name."
  • Assignee — assign all tasks to a team member. Pick from the dropdown — it shows your real ClickUp workspace members.
  • Status — set a starting status on every task. Pick from your list's actual statuses.
Don't see your custom fields? They only appear after you've selected a list in the previous step. Go back and make sure a list is selected.
7
Review and run

The final screen shows a summary of everything — your list, task count, date range, naming format, and settings. Take a quick look to make sure it all looks right.

You'll also see a preview of exactly what the first task will be named, so there are no surprises.

When you're ready, click Create Tasks in ClickUp. The app will create each task one by one — you'll see a live log as they appear. When it's done, head to your ClickUp list to see them.

Important: Running the app twice for the same date range will create duplicate tasks — it doesn't check what's already in your list first. Double-check your date range before hitting run.

Common questions

My API token isn't working
Make sure you copied the full token — it starts with pk_ and is quite long. Go back to ClickUp → Settings → Apps and copy it again. Make sure there are no extra spaces before or after it.
My lists aren't showing up when I browse
This usually means the API token wasn't entered before clicking Browse. Paste your token first, then click Browse my lists. If it's still not loading, try refreshing the page and starting again.
My custom fields aren't showing up
Custom fields only load after you've selected a list. Go back to the Connection step and make sure a list is selected — you should see the list name confirmed at the bottom of that section.
I created duplicate tasks by accident
Unfortunately the app doesn't have an undo button — you'll need to delete the duplicates in ClickUp. To avoid this in future, always check the date range and task count on the Review screen before hitting run.
I lost my license key
Email support@shinebrightvirtual.com with the email address you used to purchase and we'll resend it right away.
Something else isn't working
Email support@shinebrightvirtual.com and describe what's happening. Screenshots always help.
