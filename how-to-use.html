Before you start

Enter your license key

When you open the app for the first time, you'll see a screen asking for your license key. Enter the key that was emailed to you after purchase and click Activate.

Your key looks like: SBV-XXXX-XXXX-XXXX
Your key saves automatically — you won't need to enter it again on this browser.
Lost your key? Email support@shinebrightvirtual.com and we'll send it over.
Step 1 of 4

Create a posting schedule

On the home screen, click "+ Create a posting schedule." A 3-step modal walks you through it. Set it up once — load it every month.

1
Name it
Give it a name that describes the pattern.
Example: "Client IG - M/W/F" or "Monday Emails" or "Weekly Check-ins"
2
Set the pattern
Choose how often tasks repeat:
  • Specific days of the week (Mon, Wed, Fri)
  • Every few days (every 3 days)
  • Every few weeks
  • X posts per month, spread evenly
3
Name your tasks
Choose how each task will be titled:
  • Custom text + date → "IG Post: March 3, 2026"
  • Same title every time → "Post TBD" or "Monday Email: Topic TBD"
  • Date only → "March 3, 2026"
Click "Save schedule" when done. Your schedule appears as a card on the home screen.
Step 2 of 4

Connect to ClickUp and pick your list

Your API Token

In ClickUp: click your avatar or initials in the top left corner → Settings → Apps → copy the API Token. It starts with pk_

Paste your token in the app and it saves automatically — you won't need to enter it again as long as you use the same browser.
Finding your list
1
Click "Browse my lists"
2
Select your Workspace from the dropdown
3
Select your Space
4
Select your Folder (optional — skip if your list is not in a folder)
5
Click your List to select it
6
Or type in the search box to find any list by name
Once a list is selected, your statuses, team members, and custom fields load automatically. Nothing to look up.
Step 3 of 4

Set your dates and settings

First & last task date
Pick your start date and end date. The task count preview updates automatically — you can see exactly how many tasks will be created before you run anything.
Due date (optional)
Set the due date to match the task date, a set number of days before it, or leave it blank.
Post date custom field (optional)
If your list has a "Post Date" or "Publish Date" custom field, turn this on and select it. The date fills in automatically on every task created.
Other custom fields (optional)
Any other custom fields on your list appear here. Set a default value and it applies to every task — content type, post status, whatever you use in your setup.
Assignee & status (optional)
Assign all tasks to a team member and set a starting status. Both pull from your actual ClickUp list — just pick from the dropdown.
Step 4 of 4

Review and run

The Review screen shows a full summary of everything before any tasks are created. Take a quick look — then click Create Tasks in ClickUp.

✓
Check the task count
Does the number look right for your date range?
✓
Check the date range
Correct month and year? Jan 1 to Jan 31, not Jan 1 to Dec 31 by accident?
✓
Check the first task name
Is it formatted the way you want? The preview at the bottom of the summary shows you.
✓
Click Create Tasks in ClickUp
Tasks create one by one — you'll see them appear in a live log.
Important: running the tool twice for the same date range will create duplicate tasks. Always check your date range on the Review screen before clicking run.

Common questions

Good ones, answered.

My API token isn't working
Make sure you copied the full token — it starts with pk_ and is quite long. Go back to ClickUp, click your avatar in the top left, go to Settings → Apps, and copy it again. Make sure there are no extra spaces.
My lists aren't showing up when I browse
This usually means the API token wasn't entered before clicking Browse. Paste your token first, then click Browse my lists. If it still doesn't load, try refreshing the page.
My custom fields aren't showing
Custom fields only load after you've selected a list. Go back to the Connection step and make sure a list is selected — you'll see the list name confirmed on screen.
I created duplicate tasks by accident
Delete the duplicates directly in ClickUp. To avoid this in future, always check the date range on the Review screen before clicking run.
How is this different from ClickUp's recurring tasks?
ClickUp's recurring feature repeats a single task after it closes. Bulk Task Creator creates a full set of individual tasks upfront — each with its own due date, name, and custom fields already filled in. Your whole month is ready on day one.
Something else isn't working
Email support@shinebrightvirtual.com and describe what's happening. Screenshots always help.
