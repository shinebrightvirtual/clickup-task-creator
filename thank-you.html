Common questions
Good ones, answered.
I didn't get an email with my license key
Check your spam folder first. If it's not there, email support@shinebrightvirtual.com with your purchase email and I'll send it over. In the meantime, your key is in the dark bar above — just copy it and use it now.
My API token isn't working
Make sure you copied the full token — it starts with pk_ and is quite long. In ClickUp: click your avatar in the top left, go to Settings, then Apps, and copy it again. No extra spaces.
My lists aren't showing up when I browse
This usually means the API token wasn't entered before clicking Browse. Paste your token first, then click Browse my lists. If it still doesn't load, refresh the page and try again.
My custom fields aren't showing
Custom fields only load after you've selected a list. Go back to the Connection step and make sure a list is selected — you'll see the list name confirmed on screen.
I created duplicate tasks by accident
Delete the duplicates in ClickUp. To avoid it next time, always check the date range on the Review screen before clicking run — the task count preview shows exactly how many are about to be created.
How is this different from ClickUp's recurring tasks?
ClickUp's recurring feature repeats a single task after it closes. Bulk Task Creator creates a full set of individual tasks upfront — each with its own due date, name, and custom fields already filled in. Your whole month is ready on day one.
Something else isn't working
Email support@shinebrightvirtual.com and describe what's happening. Screenshots always help.
