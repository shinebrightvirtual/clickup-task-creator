Common questions

Good ones, answered.

I didn't get an email with my license key
Check your spam folder first — it can end up there. If it's not there either, email support@shinebrightvirtual.com with the email you used at checkout and I'll send it over. In the meantime, your key is shown above — feel free to copy and use it now.
My API token isn't working
Make sure you copied the full token — it starts with pk_ and is quite long. In ClickUp: click your avatar in the top left corner, go to Settings, then Apps, and copy it again. Make sure there are no extra spaces before or after it.
My lists aren't showing up when I browse
This usually means the API token wasn't entered before clicking Browse. Paste your token first, then click Browse my lists. If it still doesn't load, try refreshing the page and starting again.
My custom fields aren't showing
Custom fields only load after you've selected a list. Go back to the Connection step and make sure a list is selected — you'll see the list name confirmed on screen once it's set.
I created duplicate tasks by accident
Delete the duplicates directly in ClickUp. To avoid this in future, always check the date range on the Review screen before clicking run. The task count preview will tell you exactly how many are about to be created.
How is this different from ClickUp's recurring tasks?
ClickUp's recurring feature repeats a single task after it closes. Bulk Task Creator creates a full set of individual tasks upfront — each with its own due date, name, and custom fields already filled in. Your whole month or quarter is ready on day one, not one task at a time.
Something else isn't working
Email support@shinebrightvirtual.com and describe what's happening. Screenshots always help and I'll get back to you.
