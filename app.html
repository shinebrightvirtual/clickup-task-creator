Before you start

Enter your API Token to connect to ClickUp, then browse to the list where you want tasks created.

In ClickUp: click your avatar or initials in the top left corner -> Settings -> Apps -> copy the API Token. It starts with pk_

Your data is safe

Your API token goes directly from your browser to ClickUp. It is never sent to or stored on any Shine Bright Virtual server — we never see it, touch it, or log it.
