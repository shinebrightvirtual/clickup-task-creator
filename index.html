Before you start

You'll need two things from ClickUp: your API Token (a password that lets this tool talk to your account) and the List ID (which tells it where to create the tasks). Both are easy to find — see the hints below each field.

In ClickUp: click your avatar (bottom left) → Settings → Apps → copy the API Token. It starts with pk_
Open the ClickUp list where you want the tasks created. Look at the URL — the List ID is the long number after /li/ in the address bar.
Your data is safe

Your API token goes directly from your browser to ClickUp. It is never sent to or stored on any Shine Bright Virtual server — we never see it, touch it, or log it.

What is this?

If your ClickUp list has a custom date field — like a "Post Date" or "Publish Date" field — this tool can automatically fill it in with the scheduled date for each task. This is separate from the due date above.
