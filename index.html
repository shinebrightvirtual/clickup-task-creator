Questions
Is this a subscription?
No — it's a one-time purchase. Pay once and use it forever, including all future updates.
Does it work with any ClickUp plan?
Yes. Bulk Task Creator uses ClickUp's free public API, which works with all ClickUp plans including Free Forever.
Is my ClickUp data safe?
Your API token goes directly from your browser to ClickUp. It is never sent to, stored on, or seen by any Shine Bright Virtual server.
Can I use it on multiple browsers or computers?
Yes. Your license key works on any browser, any device. Just enter it once per browser and it's saved for future visits.
What if I lose my license key?
No problem — email hello@shinebrightvirtual.com with your purchase email and we'll resend it right away.
Will this create duplicate tasks?
The tool creates new tasks each time you run it — it doesn't check for existing tasks first. Running it twice for the same date range will create duplicates, so double-check your settings before hitting run.
What platforms does it work on besides ClickUp?
Right now it's built specifically for ClickUp. Support for other platforms is on the roadmap.
